## 🔧 驱动来源分类

### 1️⃣ 自行编译（GitHub Actions 自动构建）

**构建仓库**: [sunbos/Yoga-9-15IMH5-Hackintosh](https://github.com/sunbos/Yoga-9-15IMH5-Hackintosh)

**构建频率**: 每周一自动检查上游更新并编译

**构建流程**:
1. 检查上游仓库最新 commit
2. 从 master 分支克隆源码
3. 使用 xcodebuild 编译
4. 最小化处理（移除不必要的资源）
5. 上传到 GitHub Release

| 组件 | 上游仓库 | 编译说明 |
|------|----------|----------|
| **Lilu** | [acidanthera/Lilu](https://github.com/acidanthera/Lilu) | 核心依赖，最先编译 |
| **AppleALC** | [acidanthera/AppleALC](https://github.com/acidanthera/AppleALC) | 声卡驱动，仅保留 ALC298 layout-id 98 |
| **AirportItlwm** | [OpenIntelWireless/itlwm](https://github.com/OpenIntelWireless/itlwm) | Intel WiFi 驱动，针对 Ventura API |
| **IntelBluetoothFirmware** | [OpenIntelWireless/IntelBluetoothFirmware](https://github.com/OpenIntelWireless/IntelBluetoothFirmware) | Intel 蓝牙固件加载 |
| **IntelBTPatcher** | [OpenIntelWireless/IntelBluetoothFirmware](https://github.com/OpenIntelWireless/IntelBluetoothFirmware) | Intel 蓝牙运行时补丁 |
| **YogaSMC** | [zhen-zen/YogaSMC](https://github.com/zhen-zen/YogaSMC) | Yoga 特性支持（键盘灯、传感器等） |

**关键配置文件**:
- `config/device-config.json` - 设备配置（PCI ID、codec、patches）
- `last-build-sha.json` - 记录上次构建的 SHA
- `scripts/check-updates.py` - 更新检查脚本
- `scripts/minimize-*.py` - 最小化处理脚本
